# Pizza Full-Stack Application

A full-stack web application for managing a pizzeria menu with a client-server architecture.

## Tech Stack

* **Frontend:** HTML5, CSS3, JavaScript (ES6+)
* **Backend:** Python, FastAPI
* **Database:** SQL Database
* **Deployment:** GitHub

## Features

* Responsive restaurant website
* Dynamic menu loaded from the backend
* Admin panel for menu management
* CRUD operations (Create, Read, Update, Delete)
* REST API communication via Fetch API
* Input validation and error handling

## Architecture

* **Frontend:** Vanilla JavaScript
* **Backend:** FastAPI
* **Communication:** REST API (GET, POST, PUT, DELETE)
* **Database:** SQL

## Purpose

This project demonstrates the development of a small full-stack application with a clear separation between frontend and backend.
